Unlink the author link from comments.

== Description ==

This plugin, when activated, disables the author link from comments.

== Installation ==

Upload the plugin, activate it, now you're done!

== Frequently Asked Questions ==

= How to disable author link from comments? =

Activate the plugin. That's it.

= How to get back author link in comments? =

Deactivate the plugin to get it back.
